## 1. 브랜드 아이덴티티 개요

### 브랜드 키워드

```
감성 · 힐링 · 설렘 · 레트로 · 시티팝 · 밤 · 꽃 · 필름
```

### 브랜드 페르소나

> 판타로즈는 새벽 2시, 잠이 안 와서 켠 플레이리스트 같은 브랜드다.
> 시티팝이 흐르고, 창밖엔 불빛이 가득하고, 왠지 오늘 밤은 뭔가 좋은 일이 생길 것 같은 기분.

| 속성 | 내용 |
|------|------|
| **톤** | 따뜻하지만 약간 쓸쓸한. 설레지만 조용한. |
| **감성** | 레트로 시티팝 + 현대 감성 힐링 |
| **무드** | 밤 + 도시 + 꽃집 + 암실 + 편의점 불빛 |
| **타겟 감정** | "왠지 오늘 좋은 일이 생길 것 같아" |

### 브랜드가 절대 하지 않는 것

- 과한 밝음 / 과한 어두움
- 자극적인 색상 조합
- 빠른 컷 편집 (느리고 여유있게)
- 클릭베이트 제목

## 2. 컬러 시스템

### 메인 팔레트

| 이름 | 헥스코드 | 용도 |
|------|---------|------|
| **판타로즈 핑크** | `#FFB3A7` | 라티 대표색, 메인 포인트 |
| **미드나잇 퍼플** | `#7B5EA7` | 루미 대표색, 서브 포인트 |
| **크림 화이트** | `#FFF8F0` | 배경 기본, 텍스트 영역 |
| **시티 네이비** | `#2C3E6B` | 야경 배경, 딥 포인트 |
| **피피 그레이** | `#9EA3B0` | 피피 대표색, 중립 영역 |

### 서브 팔레트

| 이름 | 헥스코드 | 용도 |
|------|---------|------|
| **민트 포인트** | `#A8E6CF` | 라티 포인트 (꽃집 디테일) |
| **라벤더** | `#C9B8E8` | 피피 밴드색, 부드러운 포인트 |
| **골드 노을** | `#FFD580` | 계절 스페셜, 따뜻한 조명 |
| **암실 레드** | `#C0392B` | 루미 암실 조명, 드라마틱 포인트 |
| **필름 베이지** | `#E8DCC8` | 필름 감성 배경, 레트로 느낌 |

### 컬러 사용 원칙

**배경 계층**
```
1순위: 크림 화이트 (#FFF8F0) — 전체 기본 배경
2순위: 시티 네이비 (#2C3E6B) — 야경 / 밤 장면
3순위: 필름 베이지 (#E8DCC8) — 루미 필름 콘텐츠
```

**콘텐츠 계층**
```
메인 포인트: 판타로즈 핑크 or 미드나잇 퍼플 (동시 사용 금지)
서브 포인트: 라벤더 or 민트 (각 캐릭터에 맞게)
텍스트: 네이비 or 크림 (배경 반전)
```

**절대 금지 조합**
- 핑크 + 파란 계열 동시 메인 사용
- 레드 + 그린 조합
- 형광색 계열 전체

## 3. 타이포그래피

### 주요 폰트 시스템

| 용도 | 폰트 | 특징 |
|------|------|------|
| **헤드라인** | Noto Serif KR (한글) / Playfair Display (영문) | 감성적, 고급스러운 세리프 |
| **본문** | Noto Sans KR (한글) / Inter (영문) | 가독성 최우선 |
| **힐링 문구** | 나눔손글씨 예쁜 (한글) / Caveat (영문) | 손글씨 스타일, 감성 강조 |
| **캐릭터 대사** | 나눔고딕 (한글) | 친근하고 읽기 쉬운 |
| **브랜드 로고** | 커스텀 레터링 | 판타로즈 전용 |

### 힐링 문구 텍스트 스타일

```
폰트: 나눔손글씨 예쁜
크기: 에피소드 내 → 20~24px (모바일 기준)
컬러: 크림화이트 (#FFF8F0)
정렬: 중앙
자간: +50 (여유있게)
행간: 1.8
```

### 폰트 조합 예시

**에피소드 엔딩 카드**
```
힐링 문구 (손글씨 폰트, 크림화이트)
─────────────────
카드명 (Playfair Display, 소문자, 라벤더)
```

**SNS 캡션**
```
첫 줄: 짧은 훅 (Noto Serif, 볼드)
본문: 에피소드 힌트 (Noto Sans, 레귤러)
하단: 해시태그 (Inter, 라이트)
```

## 4. 로고 & 심볼 방향

### 브랜드 로고 컨셉

**메인 로고**
- 텍스트 로고: "판타로즈" + "Fantarose"
- 한글과 영문을 함께 쌓은 구조
- 폰트 방향: 레트로 느낌의 손글씨 + 세리프 혼합
- 컬러: 기본(핑크+ 네이비) / 화이트버전 / 다크버전

**심볼 마크**
- 장미꽃 + 별 모양 눈동자 조합
- 라티의 별 눈 + 장미 꽃잎이 합쳐진 형태
- 단독 심볼로도 사용 가능 (앱 아이콘 등)

### 로고 사용 규칙

| 상황 | 사용 버전 |
|------|---------|
| 밝은 배경 | 핑크+네이비 풀컬러 버전 |
| 어두운 배경 | 크림화이트 단색 버전 |
| 앱 아이콘 | 심볼 마크 단독 |
| 굿즈 프린트 | 상황에 맞게 선택 |

## 5. 비주얼 레퍼런스 & 무드보드

### 시티팝 비주얼 레퍼런스

**색감 레퍼런스**
- 80년대 일본 시티팝 앨범 아트 (야마시타 타츠로, 다케우치 마리야)
- 필름 카메라로 찍은 도시 야경 (약간 빛번짐)
- 레트로 네온사인 + 따뜻한 가로등

**구도 레퍼런스**
- 로우앵글에서 올려본 도시 빌딩 + 별
- 편의점 냉장고 안쪽 시점
- 꽃집 유리창 너머 도시 (아웃포커싱)
- 암실 빨간 조명 아래 사진 현상

### 애니메이션 비주얼 참고

| 레퍼런스 작품 | 참고 요소 |
|-------------|---------|
| 귀멸의 칼날 | 수채화 질감 배경, 빛 표현 |
| 스즈메의 문단속 | 현대 도시 배경 + 감성적 색감 |
| 요리이야기 | 따뜻한 실내 조명, 음식/꽃 디테일 |
| 유미의 세포들 | 캐릭터 감정 표현, 의인화 |

### 무드 키워드별 비주얼

| 무드 | 비주얼 요소 |
|------|-----------|
| **새벽 편의점** | 파란 어둠 + 편의점 형광등 노란빛 + 김서린 유리 |
| **꽃집 일상** | 온기 있는 노란 조명 + 흐드러진 꽃들 + 낡은 나무 카운터 |
| **루미 암실** | 빨간 조명 + 현상 중인 필름 + 작은 빛 포인트 |
| **시티팝 거리** | 황혼 + 네온사인 + 인파 실루엣 + 빛번짐 |
| **힐링 엔딩** | 달빛 or 별빛 + 서서히 밝아지는 배경 |

## 6. 캐릭터 비주얼 가이드

### 라티 비주얼 스타일 가이드

**일러스트 방향**
- 선 굵기: 보통 (둥글고 따뜻한 느낌)
- 채색 스타일: 수채화 느낌 + 부드러운 그라디언트
- 표정 다양성: 감정 변화 많음 (최소 10종 표정 설계)
- 비율: 머리가 살짝 큰 SD에 가까운 프로포션

**필수 표정 목록**
| 표정 | 상황 |
|------|------|
| 기본 미소 | 일상 |
| 설렘 (별눈 반짝) | 루미 등장 시 |
| 당황 (귀 내려감) | 예상치 못한 상황 |
| 행복 (초승달 눈) | 기쁜 일 |
| 수줍음 (앞치마로 가림) | 부끄러운 순간 |
| 걱정 (눈동자 별 흐릿) | 불안한 순간 |

---

### 루미 비주얼 스타일 가이드

**일러스트 방향**
- 선 굵기: 얇고 날카로운 느낌
- 채색 스타일: 레트로 필름 느낌 + 차가운 색감
- 표정 다양성: 변화 적음 (미세한 차이가 핵심)
- 비율: 라티보다 더 날렵한 프로포션

**루미 표정의 미세 차이 설계**
| 상태 | 귀 위치 | 꼬리 움직임 | 눈 |
|------|---------|-----------|---|
| 평상시 | 바짝 섬 | 정지 | 반쯤 감김 |
| 관심 있을 때 | 약간 앞으로 | 천천히 흔들림 | 살짝 크게 |
| 당황 | 빳빳이 섬 | 꼿꼿이 섬 | 평소와 같음 |
| 라티 앞 | 귀 끝 1cm 내려감 | 느리게 흔들 | 조금 풀림 |

---

### 피피 비주얼 스타일 가이드

**일러스트 방향**
- 선 굵기: 보통 (루미와 라티 중간)
- 채색 스타일: 쿨톤, 단색에 가까운 심플함
- 표정: 거의 변화 없음 (고개 방향, 눈 깜빡임으로만 표현)
- 비율: 실제 비둘기에 가까운 둥글고 묵직한 형태

**피피 행동 애니메이션 목록**
| 행동 | 의미 | 사용 상황 |
|------|------|---------|
| 고개 까딱 | 긍정/동의 | 라티 질문에 반응 |
| 90도 꾸벅 | 진지한 상황 인지 | 감정 고조 장면 |
| 자리 이동 | 직감으로 피함 | 눈치채는 순간 |
| 눈 한 번 깜빡 | "알고 있어요" | 클라이맥스 |
| 날개 탁 | 분위기 전환 | 웃음 포인트 |
| 부스러기 쪼기 | 기본 상태 | 모든 상황 배경 |

## 7. 콘텐츠 비주얼 템플릿

### 에피소드 릴스 템플릿

**화면 비율**: 9:16 (세로)
**해상도**: 1080 × 1920px

```
[상단 여백] 40px
[콘텐츠 영역] 1080 × 1700px
  - 메인 일러스트 or 씬
  - 자막: 하단 1/3 영역, 배경 반투명 블러
[하단 영역] 180px
  - 힐링 문구 (엔딩 시)
  - 카드명 (아주 작게)
```

### 오늘의 꽃 포스팅 템플릿

```
[배경] 크림화이트
[상단] "오늘의 꽃" 타이틀 (핑크)
[중앙] 꽃 일러스트 (라티가 들고 있는 형태)
[하단] 꽃 이름 + 꽃말
[최하단] 한 줄 힐링 메시지
```

### 루미 필름 포스팅 템플릿

```
[배경] 필름 베이지 (#E8DCC8)
[상단] 날짜 + 장소 (Playfair 세리프, 소문자)
[중앙] 필름 느낌 사진 (모서리 6% 비네팅)
[하단] 캡션 (짧고 차분한 텍스트)
[우하단] 루미 필름 카메라 심볼 워터마크
```

### 힐링 문구 카드 템플릿

```
[배경] 에피소드 마지막 장면 (30% 투명도 블러)
[중앙] 타로 카드 일러스트 (판타로즈 버전)
[카드 하단] 힐링 문구 (손글씨 폰트, 크림화이트)
[최하단] 카드명 (영문 소문자, 라벤더, 8px)
[하단 우측] 판타로즈 심볼 워터마크
```
