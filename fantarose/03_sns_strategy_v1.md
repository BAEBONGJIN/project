## 1. 플랫폼 전략 개요

| 플랫폼 | 역할 | 주요 목표 |
|--------|------|---------|
| **틱톡** | 바이럴 및 신규 유입 채널 | 강력한 0~3초 훅으로 대량 노출 + 밈(Meme) 확산 |
| **인스타그램** | 메인 베이스 (버추얼 인플루언서) | 세계관 구축 + '무물' 등 양방향 인터랙티브 소통 |
| **유튜브** | 검색 유입 및 롱폼 수익화 | 클립(쇼츠) + 긴 호흡 타로 Lofi 플레이리스트(광고/구독) |
| **디스코드/자체 커뮤니티** | 소유형 코어 팬덤 | 카톡/앱 의존도 탈피, 팬 2차 창작(UGC) 및 세계관 토론 |

### 론칭 순서

```
인스타/틱톡 쇼츠 동시 론칭(바이럴) → 인스타 무물 통한 팬덤 인터랙션 → 디스코드(소유형 커뮤니티) 결집 → 굿즈/웹앱 수익화
```

인스타그램으로 세계관과 비주얼 퀄리티를 먼저 쌓은 뒤,
쇼츠와 틱톡으로 대량 노출을 유도하는 구조.

## 2. 인스타그램 전략

### 계정 구조

**메인 계정 1개 + 캐릭터 계정 2개**

| 계정 | 컨셉 | 업로드 빈도 |
|------|------|-----------|
| **@fantarose.official** | 브랜드 메인 / 에피소드 릴스 | 주 4~5회 |
| **@lati.flower** | 라티의 꽃집 인스타 (1인칭) | 주 3회 |
| **@lumi.film** | 루미의 필름 사진 계정 (1인칭) | 주 2회 |

> 피피는 계정 없음. 다른 캐릭터 계정에서 목격담으로만 등장.

---

### 피드 설계 (메인 계정)

**컬러 톤 유지 원칙**
- 전체 피드가 **시티팝 레트로 감성**으로 통일
- 핑크 × 보라 × 크림 톤 유지
- 3~6컷마다 컬러 밸런스 체크

**피드 배치 패턴 (9컷 기준)**
```
[에피소드] [캐릭터컷] [힐링문구]
[캐릭터컷] [에피소드] [오늘의꽃]
[힐링문구] [에피소드] [피피밈]
```

---

### 릴스 포맷 상세

**에피소드 릴스 (주력)**

| 구성 | 내용 | 시간 |
|------|------|------|
| 훅 | 감성 장면 or 피피 등장 | 0~3초 |
| 전개 | 에피소드 씬, 음악 위주 | 3~40초 |
| 엔딩 | 힐링 문구 자막 페이드인 | 40~55초 |
| 쿠키 | 피피 단독 컷 | 55~60초 |

**캐릭터 소개 릴스 (론칭 초기)**
- 라티 / 루미 / 피피 각각 15초 숏 소개 릴스
- "이런 애야" 느낌으로 성격·직업·말투 한 번에
- 피피 편: 대사 없이 행동만 → 마지막 자막 "말을 할 수 있는지 없는지, 아무도 몰라요."

**힐링 문구 카드 릴스**
- 배경: 판타로즈 캐릭터 일러스트
- 텍스트: 타로 카드 힐링 문구
- 마지막 프레임: "당신에게 필요한 카드는?" → 타로 웹앱 링크 유도

---

### 스토리 운영

| 스토리 유형 | 내용 | 주기 |
|-----------|------|------|
| **실시간 고민 상담(무물)** | "오늘 하루 어땠나요?" 팬들의 실제 고민을 받아 라티가 타로/꽃으로 위로 처방 (가장 중요한 인터랙티브 요소) | 주 3회 |
| 세계관 힌트 | 피피 밴드 클로즈업, 루미 암실 등 | 주 2회 |
| 디스코드 유입 유도 | "피피의 단서를 찾은 분들은 이곳으로" | 주 1회 |
| 타로 웹앱/굿즈 링크 | "오늘의 카드 보러가기" / "블라인드 박스 힌트" | 비정기 |

## 3. 유튜브 쇼츠 전략

### 콘텐츠 방향

인스타그램 릴스와 같은 콘텐츠를 올리되, **썸네일과 제목**을 유튜브 알고리즘에 맞게 최적화.

**제목 공식**
- `"새벽 2시, 같은 음료를 집은 그 순간 ✨ #판타로즈"`
- `"말 못 한 고백을 사진으로 찍는 고양이 🎞️"`
- `"피피가 또 거기 있었다... 🕊️ #힐링 #판타로즈"`

**업로드 시간대**
- 평일: 오후 7~9시 (퇴근 후 스크롤 시간)
- 주말: 오전 10~12시

**쇼츠 → 롱폼 전환 전략**
- 팔로워 5만 이상: 에피소드를 3~5분 풀버전으로 유튜브 롱폼 업로드 시작
- 쇼츠는 '예고편', 롱폼은 '풀에피소드' 구조

## 4. 틱톡 전략

### 틱톡 특화 콘텐츠

**밈 컷 시리즈 (15~20초)**
- 피피의 무표정 + 엉뚱한 타이밍 등장 클립만 편집
- "피피가 거기 있었던 이유" 시리즈
- 텍스트 오버레이: "이게 왜 웃기지" / "피피 또..." 

**"당신에게 필요한 카드는?" 챌린지**
- 짧은 상황 제시 → 타로 카드 공개 → 힐링 문구
- 팔로워들이 댓글에 자기 상황을 쓰게 유도
- 인기 댓글 상황으로 다음 에피소드 제작

**루미 × 라티 설렘 컷 (30초)**
- 설레는 장면만 편집해서 올림
- 음악: 틱톡 트렌드 사운드 + 시티팝 믹스

### 틱톡 해시태그 전략

```
#판타로즈 #fantarose          ← 브랜드 고유 태그
#힐링 #감성 #시티팝           ← 장르 태그
#타로 #타로카드 #힐링타로      ← 타겟 커뮤니티
#숏폼애니 #일상캐릭터          ← 포맷 태그
#피피 #라티 #루미             ← 캐릭터 태그
```

## 5. 계정 운영 가이드

### 라티 계정 (@lati.flower) 운영 가이드

**말투**: 1인칭, 친근하고 따뜻함. 문장 끝이 올라가는 느낌.
**콘텐츠 포맷**:

```
📷 오늘의 꽃: [꽃 이름]

[꽃말 한 줄]

오늘 가게에서 있었던 일이에요.
[2~3줄 에피소드 힌트]

#오늘의꽃 #꽃말 #판타로즈 #라티
```

**운영 규칙**
- 루미 직접 언급 금지 (간접적으로만: "오늘 어떤 손님이...")
- 피피는 "우리 가게 비둘기"로만 언급
- 타로는 직접 언급 안 함 (힐링 문구 형태로만)

---

### 루미 계정 (@lumi.film) 운영 가이드

**말투**: 1인칭, 짧고 건조함. 감성적이지만 설명 없음.
**콘텐츠 포맷**:

```
📽 [날짜] — [장소]

[한 줄 또는 두 줄 캡션]

(사진 첨부 — 배경에 라티가 몰래 찍혀있는 컷 주기적으로)
```

**운영 규칙**
- 라티 직접 언급 금지 (사진 배경에만 등장)
- 피피: "암실에 이상한 비둘기가 있다" 정도만
- 캡션은 항상 3줄 이하
- 가끔 사진 캡션에서 감정이 살짝 새어나오는 순간 설계 (팬 반응 유도)

## 7. 알고리즘 공략법

### 인스타그램 릴스 알고리즘

**핵심 지표 (조회수에 영향)**

| 지표 | 목표 | 전략 |
|------|------|------|
| **시청 완료율** | 80% 이상 | 60초 이내 유지, 쿠키로 끝까지 보게 함 |
| **저장율** | 전체의 5% 이상 | 힐링 문구를 저장하고 싶게 디자인 |
| **공유율** | 저장율의 50% | "이 사람한테 보내야겠다" 공감 유도 |
| **댓글율** | 조회수의 1% 이상 | 타로 챌린지, 질문형 엔딩 자막 |

**첫 24시간 알고리즘 부스팅**
- 업로드 후 1시간 내 댓글에 답글 달기
- 스토리에 릴스 재공유
- 캐릭터 계정에서 메인 계정 릴스 태그

---

### 유튜브 쇼츠 알고리즘

**핵심 지표**

| 지표 | 목표 | 전략 |
|------|------|------|
| **시청 완료율** | 70% 이상 | 60초 안에 훅-전개-엔딩 완결 |
| **좋아요율** | 조회수의 3% | 감정 폭발 장면 후 자연스러운 좋아요 유도 |
| **구독 전환율** | 시청자의 0.5% | 쿠키 후 "다음 에피소드 기대하세요" |

---

### 틱톡 알고리즘

**For You Page (FYP) 진입 조건**

- 첫 3초 시청 유지율 > 70%
- 전체 시청완료율 > 50%
- 영상 길이: 15~30초가 FYP 최적

**틱톡 전용 전략**
- 트렌드 사운드 활용 (시티팝 + 현재 유행 사운드 믹스)
- 자막을 크게, 화면 중앙에 배치
- 첫 프레임에 텍스트 훅 삽입 ("이거 보고 소름돋았음")

## 8. 바이럴 설계 상세

### 피피 미스터리 바이럴

**시딩 방법**
1. 에피소드 영상 속 피피가 항상 다른 위치에 있음
2. 설명 없이 팬들이 알아채도록
3. 댓글에서 "피피 왜 거기 있어요?" 질문이 나오면 → 답글 없이 무시 (미스터리 유지)
4. 피피 밴드 기호 클로즈업이 등장하면 → "무슨 뜻이에요?" 자연 발생

**피피 관련 커뮤니티(디스코드) 유도 문구**
- "피피는 왜 항상 거기 있을까요? 단서를 찾으셨다면 팬덤 아지트(디스코드)에 공유해주세요."
- "피피가 말했다면 뭐라고 했을까요?" (댓글 및 커뮤니티 이벤트)
- "오늘 영상에서 피피 몇 번 나왔는지 찾아보세요" (재시청 유도)

---

### 힐링 문구 저장 유도

**디자인 원칙**
- 배경: 판타로즈 캐릭터 일러스트 (저화질 워터마크)
- 폰트: 감성적인 손글씨 계열
- 문구: 흰색, 중앙 정렬, 여백 충분히

**저장 유도 장치**
- "저장해두고 힘든 날 꺼내보세요" 캡션
- 스토리에서 힐링문구 카드 공유 → 하이라이트 보관
- 주기적으로 "어떤 문구가 가장 기억에 남으세요?" 투표

---

### 타로 챌린지

**운영 방식**
1. 매주 하나의 상황을 제시 ("오늘 ○○한 당신에게 필요한 카드는?")
2. 댓글에 자기 상황 적기 유도
3. 가장 공감 많은 댓글을 골라 다음 에피소드 소재로 활용
4. 선정된 팬에게 피피 굿즈 증정 (이벤트)

**챌린지 예시**
- "오늘 연락하고 싶은 사람이 있는 당신에게 필요한 카드는?"
- "새로운 시작을 앞두고 떨리는 당신에게 필요한 카드는?"
- "이유 없이 슬픈 당신에게 필요한 카드는?"

## 9. 팔로워 전환 전략

### 팔로워 → 팬 → 구매자 전환 경로

```
팔로워 (인지)
  ↓ 에피소드 감정 공감
팬 (몰입)
  ↓ 피피 미스터리, 타로 챌린지 참여
커뮤니티 (소속감)
  ↓ 굿즈, 타로 웹앱
구매자 (수익화)
```

### 단계별 전략

| 단계 | 팔로워 수 | 핵심 전략 |
|------|---------|---------|
| **인지** | 0 → 1만 | 콘텐츠 퀄리티 + 일관성. 무조건 쌓기 |
| **공감** | 1만 → 5만 | 타로 챌린지, 피피 이론 커뮤니티 형성 |
| **소속감** | 5만 → 10만 | 팬 참여형 에피소드, 굿즈 예고 |
| **수익화** | 10만 이상 | 굿즈 판매, 타로 웹앱 론칭, 브랜드 콜라보 |
