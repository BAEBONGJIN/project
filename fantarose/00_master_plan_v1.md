# 🌹 판타로즈 (Fantarose) — IP 종합 기획안 v1

> *"오늘 밤, 편의점 냉장고 앞에서 마주친 그 눈빛. 어쩌면 그게 시작이었을지도."*

---

## PART 1. IP 전략 (`/ip-plan`)

### 1-1. 로그라인

> 시티팝이 흐르는 도시, 꽃을 파는 분홍 토끼와 필름을 찍는 보라 고양이, 그리고 아무도 해독하지 못한 회색 비둘기가 — 말 없이 치유되는 이야기.

---

### 1-2. 시장 분석

| 항목 | 내용 |
|------|------|
| **타겟** | 10~20대 여성 메인, Z·Alpha세대 세컨더리 |
| **플랫폼** | 인스타그램 릴스 → 유튜브 쇼츠 → 틱톡 |
| **트렌드** | 힐링·로맨스 감성 숏폼 + 의인화 캐릭터 IP (유미의 세포들, 산리오계열) |
| **경쟁 차별점** | 타로 서사 연동 · 시티팝 감성 · 비언어 캐릭터 피피의 미스터리 |
| **핵심 수익 모델** | 굿즈·MD → 타로 웹앱 → 브랜드 콜라보 → OTT 피칭 |

**경쟁작 분석 & 차별점**

| 레퍼런스 IP | 공통점 | 판타로즈의 차별점 |
|-----------|--------|-----------------|
| 유미의 세포들 | 의인화, 감성 일상 | 타로 세계관 + 시티팝 감성 |
| 커비 / 산리오 | 귀여운 캐릭터 굿즈 | 로맨스 서사 + 성인 감성 |
| 마루코는 아홉살 | 일상 에피소드형 | SNS 숏폼 최적화 + 타로 연동 |

---

### 1-3. IP 핵심 개요

- **제목**: 판타로즈 (Fantarose)
- **장르**: 감성 힐링 일상 + 슬로우 로맨스 + 시티팝
- **포맷**: SNS 숏폼 시리즈 (15~60초) → 유튜브 미니 에피소드 (3~5분)
- **핵심 차별점**:
  - 에피소드가 끝날 때마다 "당신에게 필요한 카드는?" 타로 힐링 문구로 마무리 → 자연스러운 타로 웹앱 유입
  - 대사 없는 캐릭터 피피의 존재감 — 말보다 강한 감정 전달, 시청자가 해석하는 재미
  - 시티팝 사운드 + 레트로 컬러 감성으로 시각적 즉시 호감

## PART 2. 캐릭터 설계 (`/character`)

### 2-1. 라티 (Lati) ♀ — 꽃집 플로리스트

**외형 키워드**: 발랄하고, 따뜻하고, 설레는

| 항목 | 내용 |
|------|------|
| **외형** | 핑크 토끼 / 별 모양 눈동자 / 볼터치 / 앞치마 항상 착용 |
| **컬러** | 코랄핑크 × 크림화이트 × 민트 포인트 |
| **성격** | 감정 표현이 즉각적. 좋으면 바로 얼굴에 나옴. 꽃처럼 상대를 피워내는 타입 |
| **핵심 욕망** | 미래 — 아직 일어나지 않은 일에 설레고 싶다 |
| **결핍** | 불안 — 새로운 것에 설레면서도 "혼자 남겨질까봐" 두려워함 |
| **상징 아이템** | 꽃다발 / 플라워 헤어핀 / 낡은 앞치마 |

**라티의 대사 톤**
- "이 꽃, 당신한테 잘 어울릴 것 같아서요."
- "오늘 왠지 좋은 일이 생길 것 같지 않아요?"
- 절대 하지 않는 말: "어차피 안 될 거야."

**SNS 퍼소나**
- 꽃집 인스타 계정 운영하는 것처럼 → 오늘의 꽃 + 오늘의 기분
- "오늘 가게에 이런 손님이 왔었어요" 형태로 에피소드 힌트 제공

---

### 2-2. 루미 (Lumi) ♂ — 필름 사진작가

**외형 키워드**: 차갑고, 조용하고, 가끔 부드러운

| 항목 | 내용 |
|------|------|
| **외형** | 보라 고양이 / 별 모양 눈동자 / 줄무늬 꼬리 / 필름 카메라 항상 소지 |
| **컬러** | 딥퍼플 × 네이비 × 실버 |
| **성격** | 도도하고 과묵. 라티 앞에서만 귀가 살짝 내려감. 인정받고 싶지만 먼저 다가가지 않음 |
| **핵심 욕망** | 과거 — 기록하고, 붙잡고 싶다. 사라지는 것들이 무섭다 |
| **결핍** | 표현 불가 — 감정이 있어도 언어로 꺼내지 못함. 그래서 사진을 찍음 |
| **상징 아이템** | 필름 카메라 / 암실 빨간 조명 / 현상된 사진들 |

**루미의 대사 톤**
- (라티에게) "... 잘 어울리네요." ← 이게 최대한의 칭찬
- "사람은 기억하려고 사진을 찍지 않아요. 잊지 않으려고 찍는 거예요."
- 절대 하지 않는 말: "좋아해요." (행동으로만 보여줌)

**SNS 퍼소나**
- 필름 사진 공유 계정 → "오늘 찍은 사진" + 한 줄 캡션 (감성적이지만 짧음)
- 가끔 배경에 라티가 찍혀있는 사진 등장 → 팬들이 찾아내는 재미

---

### 2-3. 피피 (Pipi) — 세계관 최대의 미스터리

**외형 키워드**: 무심하고, 엉뚱하고, 어쩌면 모든 걸 아는

| 항목 | 내용 |
|------|------|
| **외형** | 회색 비둘기 / 일반 눈 (별 눈 아님) / 목에 보라색 밴드 |
| **컬러** | 쿨그레이 × 라벤더 × 오프화이트 |
| **성격** | 대사 없음. 표정 변화 거의 없음. 그러나 항상 타이밍에 맞게 등장함 |
| **핵심 역할** | 감초 + 감정의 번역기 + 미스터리. 피피가 있는 곳에 힌트가 있다 |
| **결핍** | 없음 (혹은 없어 보임) — 현재만 사는 존재. 과거도 미래도 없음 |
| **상징 아이템** | 빵 부스러기 / 암실 구석 자리 / 보라색 밴드의 정체불명 刻印 |

**피피의 표현 방식**
- 대사 없이 행동으로만 표현 (고개 까딱, 꾸벅, 자리 이동)
- 에피소드에서 감정이 고조될 때 피피가 등장 → 시청자가 "피피가 알고 있다"는 느낌
- 피피의 보라색 밴드의 비밀 → 시즌 떡밥

**팬덤 설계**
- "피피가 뭔가 알고 있다" 이론 커뮤니티 자연 발생 유도
- 피피의 밴드에 새겨진 기호? — 시즌 미스터리 소재
- 피피 단독 밈 컷 설계 → "그냥 거기 있었는데 가장 분위기 있는 캐릭터" 포지션

---

### 2-4. 삼각 관계 구도

```
라티 (미래) ←→ 루미 (과거)
           ↑
         피피 (현재)
```

- **라티 × 루미**: 슬로우 로맨스. 말하지 않아도 연결되는 사람들의 이야기
- **라티 × 피피**: 꽃집의 마스코트 관계. 라티만 피피한테 이름 붙여줌
- **루미 × 피피**: 암실 공유. 피피가 루미를 가장 많이 관찰함
- **전체 구도**: 과거와 미래 사이에서 현재를 사는 피피가 두 사람을 잇는 존재

## PART 3. 스토리 설계 (`/story`)

### 3-1. 첫 만남 — 늦은 밤 편의점

> *새벽 2시. 잠이 안 와서 나온 편의점.*
> *냉장고 앞에서 같은 음료에 손이 닿았다.*
> *힐끗. 힐끗.*
> *말은 없었지만, 그날 밤 음료 맛이 평소랑 달랐다.*

**만남의 구조**: 점진적 인지 (Slow Recognition)

| 회차 | 장면 | 감정 |
|------|------|------|
| EP.0 | 편의점 냉장고 앞 — 같은 음료에 손이 닿음 / 눈 마주침 | 설렘의 씨앗 |
| EP.1 | 꽃집 앞 — 루미가 라티의 가게 앞을 지나치며 사진 찍음 | 인지 시작 |
| EP.2 | 편의점 재회 — 또 같은 음료 / 피피가 둘 사이에 끼어듦 | 웃음 + 설렘 |
| EP.3 | 꽃 한 송이 — 루미가 말 없이 꽃 한 송이를 구매 | 감정 폭발 |

### 3-3. 시즌 아크 (전체 구조)

**시즌 1**: *늦은 밤, 편의점에서 시작된 이야기*
- 발단: 편의점 첫 만남 → 같은 음료
- 전개: 꽃집-암실-거리에서 점점 마주침 증가
- 절정: 루미가 라티를 위해 무언가를 결정하는 순간
- 결말: 말 없이 함께 걷는 장면 + 피피가 보라색 밴드를 처음으로 보여주는 장면

**시즌 2 씨앗**: 피피의 정체 + 루미와 라티의 관계 심화

## PART 4. 숏폼 론칭 전략 (`/shortform`)

### 4-1. 숏폼 포맷 설계

| 플랫폼 | 길이 | 포맷 | 업로드 빈도 |
|--------|------|------|-----------|
| **인스타그램 릴스** | 30~60초 | 감성 에피소드 + 힐링 문구 엔딩 | 주 4~5회 |
| **유튜브 쇼츠** | 60초 이내 | 에피소드 클립 | 주 3~4회 |
| **틱톡** | 15~30초 | 감정 장면 클립 + 피피 밈 | 주 3회 |

---

### 4-2. 에피소드 숏폼 구조 (1편 = 30~60초)

```
[0~3초]   훅 — 감성 장면 or 피피의 등장
[3~20초]  에피소드 전개 — 말보다 장면, 배경음악으로 감정 전달
[20~40초] 감정 절정 — 라티↔루미 설레는 순간 or 피피의 타이밍
[40~55초] 엔딩 — 화면 페이드 + 힐링 문구 자막
[55~60초] 쿠키 — 피피가 혼자 뭔가 하고 있는 장면 (웃음 포인트)
```

### 4-5. 바이럴 설계

| 장치 | 설명 |
|------|------|
| **피피 이론** | "피피가 뭔가 알고 있다" 커뮤니티 자연 형성 유도 |
| **힐링 문구 저장** | 에피소드 마무리 문구를 저장하고 싶게 디자인 |
| **타로 챌린지** | "당신에게 필요한 카드는?" → 팔로워가 자신의 상황을 댓글로 적게 유도 |
| **루미 사진 속 라티 찾기** | 루미 계정 사진 속 라티가 몰래 등장 → 팬 탐색 유도 |
| **피피 밈** | 피피의 무표정 + 엉뚱한 타이밍 등장 → 밈화 |

### 4-6. 타로 웹앱 연결 전략

> 에피소드 마무리마다 자연스럽게 타로에 대한 감성적 관심을 유발 →
> "더 알고 싶다면?" 링크로 타로 웹앱 유도 (강요 없이 궁금하게)

| 단계 | 콘텐츠 | 전환 방식 |
|------|--------|---------|
| 인지 | 힐링 문구 + 카드 이미지 워터마크 | "이 카드가 궁금하다면?" |
| 관심 | "오늘 당신의 카드는?" 시리즈 | 스토리 링크로 웹앱 연결 |
| 전환 | 웹앱 내 판타로즈 캐릭터 테마 타로 | 캐릭터 굿즈 → 앱 연동 패키지 |
