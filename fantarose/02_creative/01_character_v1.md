### 외형 상세

**얼굴**
- 별 모양 눈동자 (★) — 감정에 따라 반짝임 강도가 달라짐
- 볼터치가 데일리로 있음 (흥분하면 더 진해짐)
- 귀는 살짝 안쪽으로 굽은 형태 — 수줍을 때 귀가 내려옴

**체형 & 스타일**
- 통통하고 둥근 실루엣 — 포근한 인상
- 기본 착장: 꽃집 앞치마 (오래 써서 낡음) + 플라워 헤어핀
- off-duty 룩: 레트로 시티팝 감성 — 통넓은 바지 + 크롭 니트

**컬러 팔레트**

|------|------|
| 메인 | 코랄핑크 `#FFB3A7` |

| 볼터치 | 피치 `#FFCBA4` |

### 내면 설계

**핵심 욕망 (Want)**
> "아직 오지 않은 좋은 일이 있을 것 같아서, 오늘도 설레고 싶다."

미래를 향한 기대감이 삶의 원동력. 꽃을 파는 이유도 "누군가의 좋은 날에 함께하고 싶어서."

**핵심 결핍 (Need) & 현실 공감 포인트**
> "설레는 척하지만, 사실 혼자 남겨질까봐 무섭다."

밝음의 이면에 있는 인간관계에 대한 불안. 타겟층(1020 여성)이 자주 겪는 '거절에 대한 두려움', '무리에서 소외될까 맞추는 모습'을 투영함. 먼저 다가가지만 상처받으면 티 안 나게 앓음.

**트라우마 / 원동력**
- 과거에 소중한 것이 예고 없이 사라진 경험 (구체적 설정은 시즌 전개에 따라 오픈)
- 그래서 "지금 이 순간을 예쁘게 만들어 두자"는 마음으로 꽃집을 운영

**가장 두려워하는 것**
- 설레는 감정이 착각이었다는 것을 알게 되는 순간
- "내가 혼자 좋아한 거였나?" 하는 의심과 인간관계의 단절

| **당황** | 귀가 확 내려옴 / 손에 들고 있던 꽃을 떨어뜨림 |
| **기쁨** | 제자리에서 폴짝 뜀 / 눈이 초승달 모양으로 구겨짐 |
| **슬픔** | 표현 안 하려고 억지로 꽃 정리함 / 눈동자 별이 흐릿해짐 |
| **부끄러움** | 앞치마로 얼굴 가림 / 돌아섬 |

### 대사 톤 & 말버릇

**말투 특징**
- 문장 끝을 살짝 올리는 경향 ("...이지 않아요?" 형식)
- 생각 없이 진심이 먼저 나옴 — 필터 없는 솔직함

**대표 대사**
- *"이 꽃, 당신한테 잘 어울릴 것 같아서요."*
- *"오늘 왠지 좋은 일이 생길 것 같지 않아요?"*
- *"...루미 씨가 좋아하는 꽃이 뭔지 아직 모르네요."* (독백)

**절대 하지 않는 말**
- "어차피 안 될 거야."
- 체념이나 포기를 드러내는 표현

## 2. 루미 (Lumi)

### 기본 프로필

| 항목 | 내용 |
|------|------|
| **종족** | 보라 고양이 |
| **성별** | 남 (♂) |
| **직업** | 필름 사진작가 (독립 작업자, 암실 보유) |
| **핵심 가치** | 과거 |
| **키워드** | 기록 / 표현 불가 / 섬세함 / 거리두기 |

---

### 외형 상세

**얼굴**
- 별 모양 눈동자 (★) — 감정이 없을 때도 눈이 반짝여서 읽기 어려움
- 기본 표정: 약간 내려다보는 무표정
- 귀는 항상 바짝 세워져 있음 — 라티 앞에서만 귀 끝이 살짝 내려옴

**체형 & 스타일**
- 날렵하고 길쭉한 실루엣 — 차가운 인상
- 줄무늬 꼬리 (보라+회 교차) — 감정에 따라 꼬리 움직임이 달라짐
- 기본 착장: 오버핏 코트 + 필름 카메라 목에 항상 걸림
- 시티팝 감성: 빈티지 재킷 + 테이프드 팬츠

**컬러 팔레트**

| 용도 | 색상 |
|------|------|
| 메인 | 딥 퍼플 `#7B5EA7` |
| 서브 | 네이비 `#2C3E6B` |
| 포인트 | 실버 `#C0C0C0` |
| 꼬리 줄무늬 | 라벤더 `#C9B8E8` |

---

### 내면 설계

**핵심 욕망 (Want)**
> "사라지는 것들을 붙잡고 싶다. 그래서 사진을 찍는다."

과거에 집착하는 이유는 상실감. 기억이 흐릿해지는 것이 무서워서 모든 것을 기록함.

**핵심 결핍 (Need) & 현실 공감 포인트**
> "사진 속에 담을 게 아니라, 지금 이 순간을 살아야 한다."

과거를 붙잡으려다 현재를 온전히 느끼지 못하는 패턴. 타겟층이 겪는 '미래가 불안해 지나간 과거의 좋았던 추억에만 매달리는 번아웃/무기력증'을 투영함. 라티가 이를 처음으로 깨닫게 해주는 존재.

**트라우마 / 원동력**
- 중요한 순간을 기록하지 못해서 기억을 잃어버린 경험 (구체 설정은 시즌 전개에 따라)
- 그 이후로 셔터를 누르지 않으면 불안함

**가장 두려워하는 것**
- 감정을 표현했을 때 상대가 떠나는 것
- "말하면 달라진다"는 믿음 → 그래서 계속 말을 아낌

---

### 감정 표현 레퍼토리

| 감정 | 표현 방식 |
|------|---------|
| **흥미** | 카메라를 들었다 내렸다 반복 / 시선이 멈춤 |
| **당황** | 꼬리가 빳빳이 섬 / 평소보다 말이 더 없어짐 |
| **기쁨** | 겉으로 안 티남 / 귀 끝이 1cm 내려가는 게 전부 |
| **화남** | 말 없이 자리를 뜸 / 카메라 렌즈 캡을 닫음 |
| **설렘** | 라티를 찍다가 필름을 낭비함 (본인만 모름) |

---

### 대사 톤 & 말버릇

**말투 특징**
- 문장이 짧고 끊김 ("...그렇군요." / "됐어요.")
- 생략이 많음 — 가장 중요한 말을 항상 빼놓음
- 라티에게 말할 때만 문장이 살짝 길어짐

**대표 대사**
- *(라티에게) "...잘 어울리네요."* ← 이게 최대한의 칭찬
- *"사람은 기억하려고 사진을 찍지 않아요. 잊지 않으려고 찍는 거예요."*
- *(독백) "왜 자꾸 꽃집 앞을 지나치게 되는지 모르겠다."*

**절대 하지 않는 말**
- "좋아해요." (행동으로만 보여줌)
- 감정을 직접적으로 꺼내는 표현

| **시즌 1 후반** | 처음으로 감정을 언어로 꺼내는 순간 (가장 짧은 문장으로) |

## 3. 피피 (Pipi)

### 기본 프로필

| 항목 | 내용 |
|------|------|
| **종족** | 회색 비둘기 |
| **성별** | 미정 (성별 개념이 있는지도 불명확) |
| **직업** | 없음 (꽃집 부스러기 + 암실 낮잠 + 거리 배회) |
| **핵심 가치** | 현재 |
| **키워드** | 무심 / 미스터리 / 현재 / 감정 번역기 |

---

### 외형 상세

**얼굴**
- 일반 눈 (별 눈 아님) — 라티·루미와 다른 존재임을 시각적으로 구분
- 표정 변화 거의 없음 — 항상 0.5의 표정
- 가끔 고개를 갸웃하는 것이 최대 리액션

**체형 & 스타일**
- 통통한 비둘기 체형 그대로 — 귀엽고 묵직한 실루엣
- 목에 보라색 밴드 착용 (탈착 불가능해 보임 / 정체불명)
- 아무것도 안 들고 다님 — 그냥 있음

**컬러 팔레트**

| 용도 | 색상 |
|------|------|
| 메인 | 쿨 그레이 `#9EA3B0` |
| 서브 | 오프 화이트 `#F5F5F5` |
| 밴드 | 라벤더 `#B39DDB` |
| 눈 | 다크 그레이 `#4A4A4A` |

---

### 존재 방식

**피피의 법칙**
1. 중요한 감정적 순간에 반드시 등장
2. 아무것도 말하지 않는다
3. 그런데 가장 정확한 곳에 있다

**행동 표현 목록**
- 고개 까딱 = 긍정 / 동의
- 꾸벅 (90도) = 진지한 상황 인지
- 자리 이동 = "여기 있으면 안 될 것 같은" 직감
- 눈 한 번 깜빡 = "알고 있어요"
- 날개를 한 번 탁 침 = 분위기 전환 시도 (개그 포인트)

---

### 피피 미스터리 시스템

> 피피는 단순한 마스코트가 아닐 수 있다. 시즌이 진행될수록 조금씩 드러나는 단서들.

**떡밥 레이어**

| 레이어 | 내용 | 공개 시점 |
|--------|------|---------|
| **1차** | 피피가 항상 타이밍에 맞게 등장함 | 시즌 1 초반, 시청자가 눈치채도록 |
| **2차** | 보라색 밴드에 기호가 새겨져 있음 | 시즌 1 중반, 클로즈업 컷 |
| **3차** | 피피가 가끔 루미의 사진 속에서 사라져 있음 | 시즌 1 후반, 팬이 발견하도록 |
| **4차** | 피피의 밴드 기호가 타로 카드와 연결됨 | 시즌 2 오픈 |

**팬덤 유도 문구 (SNS 시딩용)**
- "피피는 왜 항상 거기 있을까요?"
- "오늘 영상에서 피피 밴드 잘 보셨나요?"
- "피피가 말을 안 하는 건지, 못 하는 건지 — 아무도 몰라요."

---

### 피피의 역할별 기능

| 역할 | 설명 |
|------|------|
| **감초** | 진지한 장면에 등장해 분위기를 무너뜨리는 존재 (웃음 포인트) |
| **감정 번역기** | 루미가 말 못 하는 감정을 피피의 행동이 대신 표현 |
| **서사 연결자** | 꽃집과 암실, 두 공간을 오가며 두 캐릭터를 간접 연결 |
| **미스터리** | 시청자가 "피피가 뭔가 알고 있다"고 느끼게 하는 장치 |

## 4. 삼각 관계 설계

### 관계 구도도

```
        [ 라티 ]
       ↗         ↖
  꽃집 마스코트    슬로우 로맨스
     ↙               ↘
[ 피피 ] ←암실공유→ [ 루미 ]
  (현재)              (과거)
     ↑           ↑
  라티만 이름 줌   피피가 루미를 가장 많이 관찰
```

### 관계별 상세

**라티 × 루미 — 슬로우 로맨스**

| 단계 | 상황 | 감정 |
|------|------|------|
| 첫 만남 | 편의점 같은 음료 | 설렘의 씨앗 (둘 다 인지 못함) |
| 인지 | 루미가 꽃집 앞 사진 찍다 눈 마주침 | 라티: 설렘 / 루미: 합리화 |
| 관심 | 루미가 꽃을 사러 옴 (말 없이) | 라티: 확신 / 루미: 부정 |
| 감정 절정 | 루미가 처음으로 라티를 위해 무언가를 결정 | 둘 다 감정 폭발 |
| 표현 | 루미가 처음으로 말로 꺼냄 (가장 짧은 문장) | 라티: 눈물 / 시청자: 폭발 |

**라티 × 피피 — 꽃집 마스코트 관계**
- 라티가 피피를 처음 발견하고 "피피"라고 이름 붙여줌
- 피피는 반응 없었지만 그 이후로 꽃집에 매일 옴
- 라티만 피피한테 말을 건넴 ("오늘 기분 어때?" 물어보고 혼자 대답함)
- 피피가 라티의 감정 상태를 가장 먼저 읽는 존재

**루미 × 피피 — 암실 동거**
- 피피가 루미의 암실 한 구석에서 낮잠을 자고 있었음 (언제부터인지 모름)
- 루미는 쫓아내지 않음 — 말 없이 공존
- 피피가 루미의 감정이 흔들릴 때 곁에 있어줌 (아무것도 안 하지만)
- 루미가 라티 관련 필름을 현상할 때 피피가 유독 자리를 안 뜸

## 5. 캐릭터 SNS 운영 가이드

### 계정 설계

| 캐릭터 | 계정 컨셉 | 콘텐츠 방향 |
|--------|---------|------------|
| **라티** | 꽃집 플로리스트의 일상 및 '무물' | 오늘의 꽃 + 오늘의 기분 / **인스타 스토리 '무엇이든 물어보세요'로 실제 팬들의 고민 사연 받아 타로/꽃 처방해주기 (버추얼 인플루언서화)** |
| **루미** | 필름 사진작가의 포트폴리오 | 필름 사진 1장 + 짧은 캡션 / DM이나 소통은 거의 없으나 팬들이 라티를 찍은 단서를 찾게 유도 |
| **피피** | 없음 (등장은 하지만 계정 없음) | 다른 캐릭터 계정에 목격담으로 등장하거나, 팬들이 운영하는 '피피 관찰/해석 디스코드' 유도 |

### 라티 계정 콘텐츠 예시

```
📷 오늘의 꽃: 라넌큘러스

꽃말은 "매력적인 사람"이에요.
오늘 가게에 이 꽃을 오래 들여다보던 손님이 있었는데,
아마 누군가에게 주고 싶은 마음이 있으셨던 것 같아요.

말 못 한 마음도, 꽃이 대신 전해줄 수 있으니까요. 🌸
```

📽 [필름 #47 — 늦은 오후, 꽃집 앞]

**'팝마트(Pop Mart)'식 타로 블라인드 박스 전략 중심**

수익화의 핵심은 앱보다 먼저, 캐릭터 3인방과 타로 22장을 결합한 **'블라인드 박스' 마케팅**에 있습니다. 시청자들의 소유욕과 '가챠(뽑기)' 문화를 자극합니다.

| 아이템 | 설명 |
|--------|------|
| **타로카드 22장 블라인드 박스 피규어** | 메인 상품. 각 메이저 아르카나 테마의 의상을 입은 라티, 루미, 피피 피규어가 랜덤 제공됨. |
| **시크릿 에디션 (희귀 확률)** | '세계', '영웅' 등 특정 카드를 들고 있는 특별 도색 피규어. 틱톡/릴스 등에서 엄청난 '언박싱 바이럴'을 유도. |
