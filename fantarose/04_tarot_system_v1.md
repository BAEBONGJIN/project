## 1. 타로 시스템 개요

### 핵심 철학

> **판타로즈의 타로는 "점"이 아니라 "거울"이다.**
> 미래를 알려주는 게 아니라, 지금 내 마음을 비춰주는 도구.

에피소드 안에서는 타로를 절대 언급하지 않는다.
에피소드가 끝난 뒤, 그 상황에 어울리는 카드와 힐링 문구가 등장한다.
시청자는 "타로를 배운다"가 아니라 "나도 저런 기분인데..." 하고 공감한다.

### 타로 연동 3단계

```
1단계: 에피소드로 감정 공감 (타로 언급 없음)
      ↓
2단계: 힐링 문구 + 카드 이미지로 자연스러운 타로 인지
      ↓
3단계: "이 카드가 더 궁금하다면?" → 웹앱 유입
```

## 3. SNS 타로 콘텐츠 시스템

### 에피소드 엔딩 카드 포맷

**화면 구성**
```
[배경] 해당 에피소드 마지막 장면 페이드아웃
[중앙] 타로 카드 일러스트 (판타로즈 캐릭터 버전)
[하단] 힐링 문구
[최하단] 카드명 (아주 작게, 감성적 폰트)
```

**예시 (EP.0 The Fool)**
```
배경: 라티와 루미가 편의점에서 나가는 실루엣 페이드
카드: The Fool 일러스트 (라티 버전)
문구: "지금 이 순간, 그냥 걸어가도 괜찮아요."
카드명: 0. The Fool
```

---

### "오늘의 카드" 시리즈 (SNS 정기 콘텐츠)

**운영 방식**
- 매일 또는 격일: 인스타그램 스토리에 오늘의 카드 1장 공개
- "오늘 당신에게 필요한 카드는 무엇인가요?" 투표
- 카드 결과에 따라 힐링 문구 + 판타로즈 캐릭터 일러스트

**포맷**
```
스토리 1: "오늘 당신에게 어울리는 카드를 골라보세요"
          [카드 뒷면 3장] → 투표
스토리 2: 각 카드별 힐링 문구 공개
스토리 3: "더 자세히 알고 싶다면 → 링크"
```

---

### 타로 챌린지 콘텐츠

**"당신에게 필요한 카드는?" 시리즈**

공감 상황 + 카드 + 힐링 문구 3단 구성 릴스

```
[상황 제시 - 2초]
"오늘 이유 없이 슬픈 당신에게"

[카드 공개 - 3초]
판타로즈 버전 The Moon 카드 등장

[힐링 문구 - 5초]
"헤매도 돼요. 달이 있으니까."
```

**시청자 참여형**
- 댓글: "지금 어떤 상황인지 알려주세요"
- 다음 콘텐츠 소재로 활용
- 선정된 댓글 → 스토리에 "이 분께 드리는 카드는?" 시리즈

## 4. 타로 웹앱 설계

### 앱 컨셉

> **"판타로즈와 함께 오늘의 나를 만나는 타로"**
> 점이 아니라 감정을 읽는 도구. 캐릭터가 안내해주는 힐링 타로.

### 핵심 기능 (MVP)

| 기능 | 설명 |
|------|------|
| **오늘의 카드 뽑기** | 1장 뽑기 → 판타로즈 버전 카드 + 힐링 문구 |
| **3카드 스프레드** | 과거-현재-미래 3장 배열 (라티=미래, 피피=현재, 루미=과거) |
| **캐릭터 선택** | 누구에게 물어볼지 선택 (라티/루미/피피) → 각자 다른 해석 톤 |
| **저장 & 공유** | 오늘의 카드 결과를 이미지로 저장 + SNS 공유 |

### 캐릭터별 해석 톤

| 캐릭터 | 해석 스타일 | 예시 문구 |
|--------|-----------|---------|
| **라티** | 따뜻하고 설레는 톤. 미래 지향적 | "이 카드가 나왔다는 건, 좋은 일이 오고 있다는 신호예요!" |
| **루미** | 짧고 차분한 톤. 핵심만 | "...이 카드가 맞네요. 알고 있었을 거예요." |
| **피피** | 대사 없음. 카드 이미지 + 행동으로만 | (피피가 해당 카드를 발로 콕 찍는 애니메이션) |

### 3카드 스프레드 설계

```
[루미 - 과거] [피피 - 현재] [라티 - 미래]

루미: "이미 지나온 것들이에요."
피피: (현재 카드를 무표정으로 가리킴)
라티: "앞으로 이런 일이 오고 있어요!"
```

### UX 플로우

```
홈 화면
  ↓ (캐릭터 중 누구에게 물어볼지 선택)
카드 뽑기 화면
  ↓ (카드 뒷면 탭)
카드 공개 + 힐링 문구
  ↓
저장 / 공유 / 다시 뽑기
  ↓
"오늘의 에피소드 보러가기" (SNS 링크 연결)
```

### 추가 기능 (v2 이후)

| 기능 | 설명 |
|------|------|
| **타로 일기** | 날짜별로 오늘의 카드 기록 |
| **풀 스프레드** | 켈틱 크로스 10장 배열 |
| **카드 도감** | 22장 전체 판타로즈 버전 해석 열람 |
| **피피 미스터리** | 앱 내 피피 관련 숨겨진 기능 (팬 이스터에그) |

## 5. 카드 비주얼 설계

### 판타로즈 타로 카드 디자인 원칙

**카드 구성**
```
[상단] 카드 번호 + 카드명 (영문)
[중앙] 판타로즈 캐릭터 일러스트
[하단] 카드명 (한글) + 핵심 키워드
[테두리] 판타로즈 브랜드 장식 패턴
```

**캐릭터 등장 패턴**

| 카드 그룹 | 주 등장 캐릭터 |
|----------|-------------|
| 0~7번 (여정 시작) | 라티 중심 |
| 8~14번 (시련과 조화) | 루미 중심 |
| 15~21번 (변화와 완성) | 피피 중심 + 셋 등장 |

**컬러 톤**

| 카드 그룹 | 배경 톤 |
|----------|--------|
| 0~7 | 코랄핑크 계열 (라티 컬러) |
| 8~14 | 딥퍼플 계열 (루미 컬러) |
| 15~21 | 쿨그레이 → 골드 (피피 컬러 → 완성) |

---

### 시그니처 카드 3장 상세 설계

**0. The Fool — 라티 버전**
- 일러스트: 라티가 꽃다발을 들고 한 발을 내딛는 순간
- 발밑은 절벽이지만 라티는 앞을 보며 웃고 있음
- 피피가 발밑 절벽 아래에 있음 (사실은 안전하다는 암시)
- 배경: 새벽 하늘 + 시티팝 도시 실루엣

**18. The Moon — 피피 버전**
- 일러스트: 달빛 아래 편의점 앞에 피피 혼자 앉아있음
- 주변에 달빛이 반사되는 물웅덩이
- 피피의 보라 밴드가 달빛에 빛남
- 배경: 새벽 2시 도시, 아무도 없음

**21. The World — 셋 버전**
- 일러스트: 라티 + 루미 + 피피가 같은 길을 걸어가는 뒷모습
- 좌: 라티 (꽃다발), 중: 피피, 우: 루미 (카메라)
- 배경: 황금빛 노을 + 시티팝 도시

## 6. 타로 웹앱 수익화 연결

### SNS → 웹앱/굿즈 유입 경로

```
에피소드 힐링 문구
  → "이 카드가 더 궁금하다면?"
  → 인스타 계정 프로필 링크 (Linktree 등)
  → 타로 22장 블라인드 박스 굿즈 구매 OR 웹앱(베타) 체험
  → 결과 공유 및 언박싱 챌린지 (바이럴)
  → 신규 유저 유입
```

> **수익화 우선순위 안내:** 전용 앱(App)의 무리한 초기 개발은 자본 리스크가 크므로, 1차 수익화 검증은 '타로카드 블라인드 박스 굿즈'로 진행하며 앱은 장기 목표로 배치합니다.

### 수익화 모델

| 모델 | 내용 | 시기 |
|------|------|------|
| **무료 기본** | 하루 1회 카드 뽑기 무료 | 론칭부터 |
| **프리미엄** | 횟수 제한 없음 + 타로 일기 + 풀 스프레드 | 팔로워 5만 이후 |
| **캐릭터 테마** | 판타로즈 캐릭터 테마 스킨 구입 | MVP 이후 |
| **굿즈 연동** | 앱 구매자에게 한정 굿즈 코드 제공 | 굿즈 론칭 후 |

### 웹앱 론칭 타이밍

| 단계 | 팔로워 수 | 내용 |
|------|---------|------|
| 베타 예고 | 1만 | "곧 오픈돼요" 티저 + 사전 신청 |
| MVP 오픈 | 3만 | 기본 기능만 (1장 뽑기 + 공유) |
| 정식 오픈 | 5만 | 3카드 스프레드 + 캐릭터 선택 |
| 프리미엄 | 10만 | 유료 기능 오픈 |
