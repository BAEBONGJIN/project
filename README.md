# 🌹 판타로즈 (Fantarose) — 콘텐츠 자동화 & IP 기획 프로젝트

> *"오늘 밤, 편의점 냉장고 앞에서 마주친 그 눈빛. 어쩌면 그게 시작이었을지도."*

---

## 📖 프로젝트 소개

**판타로즈(Fantarose)**는 시티팝 감성의 힐링 숏폼 애니메이션 IP를 기획하고,  
SNS 콘텐츠 제작을 자동화하기 위한 스킬 & 기획 자료 레포지토리입니다.

- 꽃집 플로리스트 토끼 **라티(Lati)** 🐰
- 필름 사진작가 고양이 **루미(Lumi)** 🐱
- 세계관 최대의 미스터리 비둘기 **피피(Pipi)** 🕊️

세 캐릭터의 슬로우 로맨스와 타로 기반 힐링 세계관을 SNS 숏폼 시리즈로 풀어냅니다.

## 🃏 타로 시리즈 구조

에피소드는 **타로 메이저아르카나 22장**을 기반으로 구성됩니다.  
에피소드 내에서 타로는 일절 언급하지 않으며, 마지막에 힐링 문구 한 마디로 마무리합니다.
